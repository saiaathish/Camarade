# Product thesis

- **Primary user:** Developers and engineering teams using AI coding agents in real repositories.
- **Problem:** Agent instructions are scattered, conflicting, stale, duplicated, and difficult to verify; teams cannot prove whether more context improves work.
- **Product:** Camarade audits repository context, compiles task-specific evidence, runs controlled agent comparisons, and reports the result.
- **Unique insight:** Context quality is measurable when original and compiled instructions run under identical conditions and are judged by repository evidence.
- **Category:** Developer Tools.
- **One-line pitch:** Agent-independent CI that proves whether coding context helps or hurts.
- **North Star:** Given a repository and task, produce an evidence-backed context contract and reproducible baseline-versus-optimized proof.
