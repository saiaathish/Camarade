# Scope lock

## Build

Repository scanner; instruction parser; conflict, duplicate, and stale-reference detection; evidence collection; context compiler; agent adapter interface; Codex adapter; worktree experiment runner; deterministic evaluator; CLI; dashboard; benchmark artifacts.

## Non-goals

Accounts; billing; organizations; IDE extensions; full multi-agent implementation; permanent personal memory; rules marketplace; enterprise infrastructure.

Stage 1 freezes this boundary. Build Week implements Codex execution first without coupling core components to Codex.
